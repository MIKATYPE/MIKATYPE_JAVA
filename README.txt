美佳のタイプトレーナ Ｖｅｒ２．０６．０３ JAVA版ソースコード

　教育用のタイプ練習ソフト「美佳のタイプトレーナV2.06」のJAVA版ソースコードです。
　動作させるにはJAVAの開発環境JDKとJAVAの実行環境JREが必要です。
　UTF-8の文字コードで記載してありますのでWindowsでコンパイルするときは-encoding UTF8の指定をして下さい。
　chromebookにて動作確認済です。
　chromebookでの使用を想定してキーボードガイドは小文字で表示してあります。
　大文字で使用したいときはソースコードを変更してください。
　成績ファイルはWindows版と同じにシフトJISコードで保存しています。
　成績ファイルをUTF-8コードで保存したいときはソースコードを変更して下さい。
　このソースコードの著作権は今村二朗が所有します。
　改変、転載、他ソフトの使用など自由にお使いください。
